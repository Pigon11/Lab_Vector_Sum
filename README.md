### 4. Что распараллелено?
Процесс суммирования элементов вектора. Так как, суммирование - это операция ассоциативная (a + b + c = (a + b) + c), что позволяет разбить вектор на части, вычислить суммы частей параллельно, а затем сложить их. Это идеальная задача для параллельных вычислений.
Использован алгоритм редукции:
1. Вектор разбивается на блоки по 256 элементов
2. Каждый блок вычисляет свою частичную сумму с помощью логарифмического сложения в разделяемой памяти
3. Все частичные суммы собираются и складываются на CPU
Такой подход позволяет задействовать тысячи потоков GPU и значительно ускорить вычисления на больших векторах.
